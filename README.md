# Infocast (infocast)

Infocast is an event management company (acquired by Clarion Events in 2024)
that has produced premier deal-making conferences and summits for more than
35 years across energy, finance, infrastructure, manufacturing, and emerging
technology sectors. Infocast does not currently publish a public API.
